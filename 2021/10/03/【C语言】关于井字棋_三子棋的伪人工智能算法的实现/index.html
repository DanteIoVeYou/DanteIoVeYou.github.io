井字棋/三子棋的伪人工智能算法的实现

一、Abstract

本算法用于实现井字棋游戏电脑先手情况下的智能落子

二、Introduction

关于井字棋的模块的搭建，分为棋盘的初始化、棋盘的展示、电脑落子、玩家落子、判断输/赢/平局/继续游戏这5个模块。

其中，我们着重介绍一下电脑落子的具体算法。本算法可以使得玩家永远赢不了棋局。（输/平局）

本算法的核心思想为 暴力枚举 和 迫使玩家进行指定位置的落子
注：

  1. 本棋盘为3*3的大小
  2. 电脑先手
  3. 以下用形如(2,1)的形式表示落子的坐标
  4. 用“**#”表示电脑的棋子，用“***”表示玩家的棋子
  5. 代码部分使用C语言进行实现

  2. (1,3)或(3,1)
  3. (2,3)或(3,2)
  4. (2,2)
  5. (3,3)
